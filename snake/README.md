# Snake Game
Basic snake game with console interface.

Steps:
- Create a snake
- Create a food container
- while snake is not dead
    - let player change snake movement direction
    - let snake grow in size
    - update snake position

Example result:
```text

####.ff..f
#......f..
####....f.
##.#.ff...
```
